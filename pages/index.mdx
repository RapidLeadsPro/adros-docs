# Adros AI

**The marketing brain for AI agents.**

Adros is a pattern-driven creative intelligence platform that powers AI agents to build, launch, and optimize paid ad campaigns across Meta and Google — consistently, at enterprise quality, without human expertise.

<br />

## What Adros is

- **4,022 performance-tested ad patterns** across 17 creative styles and 45 copy frameworks — the moat
- **74 MCP tools** for pattern matching, creative generation, campaign building, memory, monitoring
- **Daily autonomous monitor** — 6 health checks running at 8am SGT on every paying account
- **Contract layer** — deterministic deliverable specs so specialists know exactly what to produce
- **Creative pipeline** — Nano Banana Pro (Gemini 2.5) generation anchored to pattern blueprints
- **Memory layer** — business context + personas + weekly action log, scoped per client

## What Adros is NOT

- Not an orchestrator — pair with Paperclip, Jarvis, or any MCP-compatible agent runtime
- Not a task ledger — pair with Paperclip or your own issue tracker
- Not a human chat interface — the dashboard is for inspection, the API is for agents

## Who this documentation is for

- **Agent builders** — integrating Adros MCP into Claude Desktop, Cursor, Paperclip, OpenClaw, or custom agents
- **Engineers** — wiring Adros into existing orchestration stacks via REST or MCP
- **Operators** — understanding the monitoring loop, webhook payloads, and contract layer

## Get started

**Sign up:** [app.adros.ai](https://app.adros.ai) → connect Meta/Google ad accounts → copy your MCP token from Settings.

**MCP endpoint:** `https://api.adros.ai/mcp` (Streamable HTTP, spec `2025-03-26`)

**Auth:** `X-User-Token: <your-mcp-token>` header on MCP connection, or `Authorization: Bearer <your-mcp-token>` for REST.

**REST base URL:** `https://api.adros.ai/api/v1`
