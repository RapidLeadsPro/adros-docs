# Adros

**Hybrid marketing OS** — built for teams who need **traceable orchestration** and **senior-marketer judgment** in the same stack.

Adros combines:

1. **Operator surfaces** at [app.adros.ai](https://app.adros.ai) — client-scoped **Command Center** (task inbox, live timeline, approvals), **Strategy Workspace** (staged artifacts and masters), **Files** (R2-backed storage), **Campaigns** (Google and Meta when connected), and **Adros CEO** (fast coordination and routing). The dashboard is not an afterthought; it is where work becomes **auditable**.

2. **Agent integration** — the same backend powers **~74 MCP tools** (Streamable HTTP) so Claude, Cursor, Paperclip-style stacks, or custom agents call the same APIs operators see.

3. **Pattern intelligence** — **4,022** performance-tested ad patterns across creative styles and copy frameworks, plus memory (business context, personas, weekly action log) scoped per client.

Plug Adros into any MCP-compatible client and the agent gets marketing execution hands on Meta/Google **and** the same task/events model the dashboard uses.

---

## What's inside

- **Command Center** — tasks as the unit of work; status, cost, specialist/agent attribution; **event timeline** for traceability; approve/reject flows; real-time task updates where enabled
- **Strategy Workspace** — lifecycle stages (research → economics → structure → copy/creative → audit → deploy) with **canonical artifacts** and promote-to-master discipline
- **Files** — uploads and browsing backed by **Cloudflare R2**, scoped by user/org/client like the rest of the API
- **Campaigns** — live performance views for **Google Ads** and **Meta Ads** when tokens and account selection are configured; scope and fallback behavior surfaced in-product
- **Quality track (audits)** — scorecards, guardrails, and persisted audit artifacts (markdown/JSON/PDF paths) on completed audits; gated by `AUDIT_QUALITY_TRACK_ENABLED` in backend env — confirm on your deployment
- **4,022 performance-tested ad patterns** — the creative moat
- **~74 MCP tools** — patterns, creative generation, campaign actions, memory, monitoring, and more
- **Daily autonomous monitor** — health checks (creative fatigue, search waste, RSA health, conversion health, Advantage+, experiments), webhook to your orchestration URL
- **Contract layer** — deterministic deliverable specs for lifecycle stages (see [Contract Layer](/integrations/contract-layer))
- **Creative pipeline** — generation anchored to pattern blueprints (e.g. Gemini image path), with product guardrails
- **Memory layer** — business context, personas, weekly logs; append-only where it matters for audit

---

## Hybrid in one sentence

**Deterministic rails** (tasks, events, files, scoped integrations) **plus** **managed-agent depth** where judgment is required — so you get speed *and* a paper trail, not a black-box chat.

## Production vs repo

Dashboard and backend features ship continuously. **What you see on `app.adros.ai` / `api.adros.ai` may lag the latest Git `main` by one deploy.** For integration testing, prefer verifying against the environment your token targets; treat this site as the **intent** of the system, validated against live OpenAPI and behavior.

## Get started in 3 minutes (MCP)

```json
{
  "mcpServers": {
    "adros": {
      "url": "https://api.adros.ai/mcp",
      "headers": { "X-User-Token": "lyr_YOUR_TOKEN" }
    }
  }
}
```

1. Sign up at [app.adros.ai](https://app.adros.ai) and copy your MCP token from **Settings → MCP**
2. Paste the JSON above into your AI client's MCP config
3. Restart the client — you should see the Adros tool set loaded

## Connection details (reference)

| Surface | Value |
|---|---|
| **MCP endpoint** | `https://api.adros.ai/mcp` |
| **MCP transport** | Streamable HTTP (spec `2025-03-26`) |
| **MCP auth header** | `X-User-Token: lyr_YOUR_TOKEN` |
| **REST base URL** | `https://api.adros.ai/api/v1` |
| **REST auth header** | `Authorization: Bearer lyr_YOUR_TOKEN` |
| **Webhook signature** | `X-Adros-Signature: sha256=<hex>` (HMAC-SHA256, `sort_keys=True` JSON) |
